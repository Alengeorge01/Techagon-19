  1. Deep Learning with Keras

    This workshop will introduce you to the field of deep learning and teach you the fundamentals. You will learn about some of the exciting applications of deep learning, the basics of neural networks, different deep learning models, and how to build your first deep learning model using the easy yet powerful library Keras.

  2. Blockchain Decentralisation

    In this workshop one will learn a practical approach to various cryptocurrencies such as bitcoins, blockchain wallets and its peer-to-peer(decentralized) properties.

  3. Robotics Process Automation

    In this workshop one shall learn to configure computer software or a “robot” to capture and interpret existing applications for processing a transaction, manipulating data, triggering responses and communicating with other digital systems.

  4. Enterprise Mobile App Developement

    Android App Development Workshop mainly focus to help the student to build Android applications using Android Studio from scratch. Students will be exposed to XML to build the front end and work on Java for the backend.

  5. Full Stack Web Development

    This worskhop will focus on training the students in both front end and back end development. Create a seamelessly created project using complete and structured knowledge in the development and deployment.

  6. Data Security and Web Hacking

    This workshop focuses on how advanced client based, server based and application based web attacks are performed in a simulated/test environment in an ethical way. One shall learn to mitigate these attack using the recommended solution at the end of each module.
